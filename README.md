## HTML Set Up
- create a html file `index.htm`
- add this bare minimum code there
```html
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Fund Me Web Page</title>
    </head>
    <body>
        hello!
    </body>
</html>
```
